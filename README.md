# 🎬 Catálogo de Filmes

Um catálogo interativo de filmes responsivo, moderno e minimalista. O projeto permite filtrar os filmes por texto e gênero, além de gerenciar uma lista de favoritos que fica salva diretamente no navegador do usuário.

📱 **Destaque do projeto:** Todo o código foi inteiramente programado e testado em um dispositivo móvel utilizando o editor de código **Acode**.

---

## 🚀 Funcionalidades

* **Filtro Inteligente:** Pesquisa em tempo real por nome do filme.
* **Filtro por Categoria:** Combinação da busca por texto com seleção de gêneros (Ação, Comédia, Ficção e Animação).
* **Sistema de Favoritos:** Permite favoritar/desfavoritar filmes com persistência de dados.
* **Persistência com LocalStorage:** Seus filmes favoritos continuam salvos mesmo se você fechar ou recarregar a página.
* **Interface Responsiva:** Layout adaptável para qualquer tamanho de tela (Grid System).

---

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estruturação semântica da página.
* **CSS3:** Estilização moderna, uso de Grid Layout e efeitos de animação (`hover`).
* **JavaScript (ES6+):** Manipulação do DOM, funções de array (`filter`, `forEach`, `includes`) e persistência com `localStorage`.
* **Acode - Powerful Code Editor:** Ambiente de desenvolvimento utilizado diretamente no smartphone.

---

## 📸 Demonstração Visual

O layout é limpo e focado na experiência do usuário, contando com um tema escuro (*dark mode*) nativo:

* **Cards de Filmes:** Imagens com efeito de *zoom* ao passar o mouse.
* **Botão de Favoritos:** Altera dinamicamente entre os estados 🤍 (Favoritar) e ❤️ (Favoritado).

---

## 💻 Como Executar o Projeto

Como o projeto foi construído puramente com tecnologias web estáticas, você não precisa instalar nenhuma dependência.

1. Faça o clone deste repositório:
   ```bash
   git clone [https://github.com/ariel-p2/Catalogo-Filmes.git](https://github.com/ariel-p2/Catalogo-Filmes.git)

2. Abra a pasta do projeto.
Dê um duplo clique no arquivo index.html para abrir diretamente no seu navegador de preferência.
3. Se estiver no celular usando o Acode, basta abrir a pasta do projeto no app e clicar no botão de "Play" (Visualizar no navegador) interno do editor.
